# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
which we summarise here so you don't have to click through.

## In short

- Be respectful. Disagree with the work, not the person.
- Assume good faith. The other side of every comment is a human trying to help.
- Keep discussion technical and on-topic. Crypto bugs need precision, not heat.

## Enforcement

The maintainer enforces this informally — most disagreements are about technical choices, not
behaviour. If you witness or experience behaviour that violates the spirit above, contact the
maintainer privately (the same channel listed in [SECURITY.md](SECURITY.md) for vulnerability
reports works).

## Scope

This Code applies in every project space: issues, PRs, discussions, code reviews, and any other
communication where someone is acting on behalf of this project.
